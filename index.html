My Research

It turns out that the world is simpler than we thought, both empirically and theoretically. So with the right tools, it becomes easy to understand, model, and control the world.

For example:

  Very simple optimizers can dramatically improve the performance of data miners learning software quality predictors.
  Text miners can succinctly summarize thousands of technical papers about SE.
  Contrast set learners find simple controllers in requirements models; reduce 6000 node trees to just 6 rules; out-perform SOA optimizers for spacecraft control; and test truisms in SE.
  Active learners can estimate large software projects after just a few samples; and reduce the cost of search-based SE by orders of magnitude.
  Clustering tools enable transfering lessons learned between software projects.
  Feature and instance selectors that let software projects share and reason about privitized data.
  Social metrics can generate near-optimal software quality predictors.
