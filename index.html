Tools for Simplicity

Turns out, the world becomes much simpler than we thought, both empirically and theoretically. So with the right tools, it becomes easy to understand, model, and control the world.

For example:

  • Very simple optimizers:
    • Can dramatically improve the performance of data miners;

  • Text miners:

    • To succinctly summarize thousands of technical papers.

  • Contrast set learners:

    • Find simple control patterns in requirements models;
    • Reduce 6000 node decision trees to six simple rules;
    • Out-perform state-of-the-art optimizers for controlling spacecraft;
    • Test truisms in software engineering;

  • Active learners:

    • That can estimate large software projects after just a few samples;
    • Reduce the cost of multi-objective optimization by orders of magnitude.

  • Clustering tools:

    • Enable transfering lessons learned between software projects;

  • Feature and instance selectors:

    • That let communities share and reason about privitized data.

  • Social metrics:

    • That generate near-optimal defect prediction;
